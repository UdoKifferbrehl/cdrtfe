Hinweise zur Nutzung der Mustangpeak Common Library, Mustangpeak EasyListview,
und Mustangpeak VirutalShellTools
==============================================================================

Einige Funktionen von cdrtfe bauen auf der Mustangpeak Common Library, dem
Mustangpeak EasyListview und den Mustangpeak VirutalShellTools auf, somit kann
das Programm nur kompiliert werden, wenn diese Komponenten installiert sind.
Download: http://code.google.com/p/mustangpeakcommonlib/
          http://code.google.com/p/mustangpeakeasylistview/
          http://code.google.com/p/mustangpeakvirtualshelltools/

Falls eine Installation nicht erwünscht ist, kann von der cdrtfe-Download-Seite
ein Archiv mit den benötigten Teilen dieser Komponenten heruntergeladen werden,
das dann in das jeweils passende Verzeichnis entpackt werden muß
(import\mpcommonlib, import\mpeasylistview, mpvirtualshelltools).


Notes on the use of the Mustangpeak Common Library, Mustangpeak EasyListview,
and Mustangpeak VirutalShellTools
=============================================================================

Some functions of cdrtfe are based on the Mustangpeak Common Library,
Mustangpeak EasyListview and Mustangpeak VirutalShellTools. Thus, these
components have to be installed in order to compile the program.
Download: http://code.google.com/p/mustangpeakcommonlib/
          http://code.google.com/p/mustangpeakeasylistview/
          http://code.google.com/p/mustangpeakvirtualshelltools/

If you don't want to install these components, you can get a package which
contains the needed parts of these components from the cdrtfe download page.
Extract the files into the approriate folder (import\mpcommonlib,
import\mpeasylistview, \import\mpvirtualshelltools).
