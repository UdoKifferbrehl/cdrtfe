Hinweise zur Nutzung der JVCL
=============================

Einige Funktionen von cdrtfe bauen auf der JVCL (JEDI Visual Component Library,
Project JEDI) auf, somit kann das Programm nur kompiliert werden, wenn die JVCL
installiert ist. Die JVCL (Quelltexte und Installationsprogramm) kann herunter-
geladen werden von http://jvcl.sourceforge.net/.

Falls eine Installation nicht erwünscht ist, kann von der cdrtfe-Download-Seite
ein Archiv mit den benötigten Teilen der JVCL heruntergeladen werden, das dann
in dieses Verzeichnis (import\jvcl_mini) entpackt werden muß.


Notes on the use of the JVCL
============================

Some functions of cdrtfe are based on the JVCL (JEDI Visual Component Library,
Project JEDI). Thus, the JVCL has to be installed in order to compile the
program. You can download the JVCL (source code and installer) from
http://jcl.sourceforge.net/.

If you don't want to install the JVCL, you can get a package which contains the
needed parts of the JVCL from the cdrtfe download page. Extract the files into
this folder (import\jvcl_mini).
