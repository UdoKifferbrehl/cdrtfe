Hinweise zur Nutzung des Virtual Treeview
=========================================

Einige Funktionen von cdrtfe bauen auf dem Virtual Treeview auf, somit kann
das Programm nur kompiliert werden, wenn diese Komponente installiert ist.
Download: http://code.google.com/p/virtual-treeview/

Falls eine Installation nicht erwünscht ist, kann von der cdrtfe-Download-Seite
ein Archiv mit den benötigten Teilen dieser Komponente heruntergeladen werden,
das dann in dieses Verzeichnis entpackt werden muß (import\virtualtreeview).


Notes on the use of the Virtual Treeview
========================================

Some functions of cdrtfe are based on the Virtual Treeview. Thus, this
component has to be installed in order to compile the program.
Download: http://code.google.com/p/virtual-treeview/

If you don't want to install this component, you can get a package which
contains the needed parts of this component from the cdrtfe download page.
Extract the files into the approriate folder (import\virtualtreeview).
