Hinweise zur Nutzung der JCL
============================

Einige Funktionen von cdrtfe bauen auf der Drag and Drop Component Suite
auf, somit kann das Programm nur kompiliert werden, wenn die Drag and 
Drop Component Suiteinstalliert ist. Die Drag and Drop Component Suite
kann heruntergeladen werden von http://melander.dk/delphi/dragdrop/.

Falls eine Installation nicht erwünscht ist, kann von der cdrtfe-Download-Seite
ein Archiv mit den benötigten Teilen der Drag and Drop Component Suite
heruntergeladen werden, das dann in dieses Verzeichnis (import\oledragdrop)
entpackt werden muß.


Notes on the use of the JCL
===========================

Some functions of cdrtfe are based on the Drag and Drop Component Suite.
Thus, the Drag and Drop Component Suite has to be installed in order to compile
the program. You can download the Drag and Drop Component Suite from
http://melander.dk/delphi/dragdrop/.

If you don't want to install the Drag and Drop Component Suite, you can get a
package which contains the needed parts of the Drag and Drop Component Suite 
from the cdrtfe download page. Extract the files into this folder
(import\oledragdrop).
