Hinweise zur Nutzung der dw Windows 7 Komponenten
=================================================

Einige Funktionen von cdrtfe bauen auf den dw Windows 7 Komponenten
auf, somit kann das Programm nur kompiliert werden, wenn diese Komponten
installiert sind. Die dw Windows 7 Komponenten können heruntergeladen werden
von http://www.gumpi.com/blog/.

Falls eine Installation nicht erwünscht ist, kann von der cdrtfe-Download-Seite
ein Archiv heruntergeladen werden, das dann in dieses Verzeichnis
(import\dwWin7Components) entpackt werden muß.


Notes on the use of the dw Windows 7 Components
===============================================

Some functions of cdrtfe are based on the dw Windows 7 Components. Thus, these
components have to be installed in order to compile the program. You can down-
load the dw Windows 7 Components from http://www.gumpi.com/blog/.

If you don't want to install these componente, you can get a package from the
cdrtfe download page. Extract the files into this folder (import\dwWin7Components).
